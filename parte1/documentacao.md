## Parte 1 - Blink LED Interno

### Código no Arduino

```
// parte 1

void setup () {
  pinMode(10, OUTPUT);
}

void loop() {
  digitalWrite(10, HIGH);
  delay(1000);
  digitalWrite(10, LOW);
  delay(1000);
}
```
